# Asset Files

The following placeholder files need to be added to this folder:

1. **icon.png** (1024x1024) - App icon
2. **splash.png** (1284x2778) - Splash screen
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon
4. **favicon.png** (48x48) - Web favicon

## Using Default Assets

For now, Expo will use default assets. You can:

1. **Option 1:** Use default assets (works fine for development)
2. **Option 2:** Add custom assets later using:
   - Any image editor to create PNG files
   - Use a logo with "#0f172a" (dark blue) background
